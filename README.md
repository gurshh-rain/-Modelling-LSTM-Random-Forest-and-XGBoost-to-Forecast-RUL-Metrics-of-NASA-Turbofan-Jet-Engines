# Predictive Maintenance Modelling Comparing LSTM, Random Forest, and XGBoost to Forecast RUL Metrics of NASA Turbofan Jet Engines
Predictive maintenance aims to estimate the Remaining Useful Life (RUL) of mechanical systems so that maintenance can be scheduled proactively rather than reactively, reducing both unplanned downtime and unnecessary servicing. This project develops and compares three modelling approaches, Random Forest, XGBoost, and a Long Short-Term Memory (LSTM) neural network, to predict the RUL of turbofan jet engines using NASA's C-MAPSS degradation dataset. Feature engineering (rolling statistics and trend features) was applied to give tree-based models access to temporal degradation patterns, while the LSTM consumed raw sliding-window sequences directly. The best individual model, the LSTM, achieved a test RMSE of 12.72 cycles and R² of 0.899; XGBoost with engineered features achieved an RMSE of 16.47 and R² of 0.831. A weighted ensemble of XGBoost and LSTM predictions was also constructed, with the blend weight tuned to minimize RMSE; the optimized ensemble achieved the best overall result of any single approach, RMSE 12.59 and R² 0.901. Results confirm that sequence-aware models capture degradation trends more effectively than single-snapshot tree-based models on this dataset, while also surfacing practical pitfalls in feature engineering and model evaluation methodology that are broadly relevant to time-series predictive maintenance work. 
Remaining Useful Life (RUL) quantifies the operational time (e.g., hours or cycles) an asset has left before it requires repair or replacement. Accurately estimating RUL allows organizations to move from reactive or fixed-interval maintenance schedules toward condition-based maintenance, where interventions are triggered by the actual health of the equipment. This reduces both the cost of premature part replacement and the risk of unexpected in-service failure, a particularly high-stakes trade-off in aerospace applications such as turbofan jet engines. 
NASA's Commercial Modular Aero-Propulsion System Simulation (C-MAPSS) dataset is a widely used benchmark for RUL prediction research. It provides multivariate time-series sensor data (temperatures, pressures, rotational speeds, and related engine health indicators) recorded across the full operational life of multiple simulated turbofan engines, from a healthy starting condition through to failure. The FD001 subset used in this project represents the simplest configuration: a single operating condition and a single fault mode, making it a suitable starting point for comparing modelling approaches before extending to the more complex FD002 to FD004 subsets. 
This project compares three models: 
■ Random Forest (tuned via GridSearchCV and RandomizedSearchCV): an ensemble of decision trees, used as a robust, low-tuning-effort baseline.
■ XGBoost: a gradient boosted tree ensemble that builds trees sequentially, with each new tree fitting the residual errors left by the current ensemble; individual tree construction is internally parallelized for computational efficiency, and the approach generally outperforms Random Forest on structured, tabular data. 
■ LSTM: a recurrent neural network architecture designed to learn from sequential data, allowing the model to natively capture degradation trends across a window of cycles rather than relying on hand-engineered temporal features. 
A secondary goal was to test whether combining the tree-based and sequence-based approaches via a weighted ensemble could outperform either model individually.
